12 - Write a JS code to find product of two arrays.

Description: Function findProd() to generate a new array which is the product of two arrays of the same size using for loop.

Example:

let arr1 = [3,45,23,78,34];
let arr2 = [4,2,34,4,12,1];
console.log(findProd(arr1,arr2)); //[12, 90, 782, 312, 408, 1]
