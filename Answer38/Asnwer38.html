38 - Find the a number is present in given range.

Description: Function `checkInRange()` finds if the given number is within the provided start and end range using >=, <= and && operators in JavaScript. 1. Print “Between the range” if num is between start and end values 2. Else Print “Outside the range” since num is outside start and end values.

Example:


checkInRange(15,11,30); //"15 is between the range 11 and 30"
checkInRange(20,34,51); //"20 is outside the range 34 and 51"
