43 - Move an array element from one position to another.

Description: Write a JavaScript function to move an array element from one position to another.

Example:


console.log(move([10, 20, 30, 40, 50], 0, 2)); //[20, 30, 10, 40, 50]
console.log(move([10, 20, 30, 40, 50], -1, -2)); //[10, 20, 30, 50, 40]
