14 - Write a JS code to find N value in the Fibonacci series for a given number.

Description: The function Findfibonacci() prints the index of number in the Fibonacci series if present or “element not present” if the number is not part of the Fibonacci series.

Example:


findFibonacci(13); //"Element 13 is present in index 7"
findFibonacci(144); //"Element 144 is present in index 12"
