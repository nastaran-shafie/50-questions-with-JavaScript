41 - Ungrouping the elements in an array produced by zip.

Description: Write a JavaScript function to create an array of arrays, ungrouping the elements in an array produced by zip.

Example:


unzip([['a', 1, true], ['b', 2, false]]); // [["a","b"],[1,2],[true,false]]
unzip([['a', 1, true], ['b', 2]]); // [["a","b"],[1,2],[true]]
